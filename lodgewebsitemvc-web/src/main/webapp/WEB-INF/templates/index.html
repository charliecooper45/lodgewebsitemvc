Welcome to the Spring MVC Lodge demo application!

A demo website created to help me learn the Spring framework (especially Spring MVC). The idea for the website came when I was creating a simple website for my parent's lodge in Cornwall (GitHub link here). Their site is static with no interaction or persistence but provided a good basis and business idea for an interactive website. The site allows users to create an account, register/manage bookings and leave feedback. The main technologies used are listed below:

  • Spring (MVC)
  • Thymeleaf (layouts managed with the Thymeleaf Layout Dialect)
  • Maven

As part of the project I decided to build and setup a Linux home server (the current distro is Ubuntu).
